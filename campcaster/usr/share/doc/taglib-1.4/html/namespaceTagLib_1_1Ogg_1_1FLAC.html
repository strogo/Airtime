Detailed Description

An implementation of Ogg FLAC metadata.

This is implementation of FLAC metadata for Ogg FLAC files. For "pure" FLAC files look under the FLAC hiearchy.

Unlike "pure" FLAC-files, Ogg FLAC only supports Xiph-comments, while the audio-properties are the same.
